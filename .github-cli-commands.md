# GitHub CLI Quick Reference

## Check CI/CD Status

```bash
# View all workflow runs
gh run list --limit 10

# View runs for current branch
gh run list --branch $(git branch --show-current)

# Watch the latest run in real-time
gh run watch

# View specific run details
gh run view RUN_ID --log

# Re-run failed workflows
gh run rerun RUN_ID
```

## Pull Requests

```bash
# Create PR from current branch
gh pr create --title "Your PR Title" --body "Description"

# List all PRs
gh pr list

# Check PR status
gh pr status

# View PR checks (CI/CD)
gh pr checks

# Merge PR
gh pr merge --squash
```

## Repository Management

```bash
# View repository
gh repo view

# Open repo in browser
gh repo view --web

# View Actions page in browser
gh browse --branch $(git branch --show-current) -- actions
```

## Workflow Management

```bash
# List all workflows
gh workflow list

# Run a workflow manually
gh workflow run WORKFLOW_NAME

# View workflow runs
gh workflow view WORKFLOW_NAME
```

## Quick Aliases (Add to ~/.bashrc or ~/.zshrc)

```bash
alias ghrl='gh run list --limit 10'
alias ghrw='gh run watch'
alias ghpl='gh pr list'
alias ghps='gh pr status'
alias ghpc='gh pr checks'
```

## Installation Verification

```bash
gh --version
gh auth status
```
