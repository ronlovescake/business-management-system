## How to Add the Clock Background

1. Save your minimalist clock image (the one with the wooden clock on a white wall) to this directory
2. Name it `minimalist-clock.jpg`
3. The login page is already configured to use this image

The image should be:

- High resolution (at least 1920x1080)
- Optimized for web (compressed to reasonable file size)
- In JPEG or WebP format
